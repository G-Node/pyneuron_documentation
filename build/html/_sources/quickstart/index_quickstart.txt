**Goals:**

#. :ref:`intro_pyneuron`

   Exercise: **Create a bipolar neuron and simulate a current clamp
   experiment**

     * Create a three compartmental model with one soma and two dendrites
     * Simulate a somatic current clamp experiment
     * Visualise results of the simulations with matplotlib

#. :ref:`intro_pydesign`

   Exercise: **Simplify the code**

     * Keep control of your code by organizing modules/libraries
     * Use object oriented programming paradigm

#. :ref:`simulation`

   Exercise: **Use a SQL database to control data**

     * Create a SQLite database
     * Run a simulation for a given parameter space
     * Store data in a database
     * Analyze data with a database
